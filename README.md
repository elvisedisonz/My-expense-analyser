# Bank Statement Analyzer

Flask-based financial transaction analyzer that:
- parses PDF and Excel bank statements
- categorizes expenses
- detects reimbursements
- groups recurring payments
- provides financial summaries

## Features

- PDF statement extraction
- Excel statement extraction
- Smart transaction grouping
- Rule-based categorization
- Reimbursement matching
- Interactive dashboard

## Tech Stack

- Python
- Flask
- Pandas
- Regex/NLP-style cleaning
- JSON rule engine
